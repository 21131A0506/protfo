Creating a Modern Portfolio with Bolt AI and Flask

Anukula Mohan Chandra Front End Developer

Introduction

In this comprehensive guide, I'll walk you through my journey of creating a modern portfolio website using Bolt AI and deploying it with Flask on PythonAnywhere. This approach combines the power of AI-assisted development with the reliability of Python web frameworks.

Getting Started with Bolt AI

Bolt AI is a powerful development assistant that helped me scaffold the initial template for my portfolio. Here's how I used it:

  • Started with a basic project structure request
  • Specified my design preferences and requirements
  • Received a complete template with HTML, CSS, and JavaScript files
  • Made iterative improvements through conversation with Bolt

Customizing the Template

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('index.html')

@app.route('/blog')
def blog():
    return render_template('blog.html')

After getting the initial template, I made several customizations:

  • Adapted the design to match my personal brand
  • Added Flask routing for dynamic content
  • Implemented a blog system
  • Enhanced the responsive design
